This API is a modular API - if you will use the type "biz_id" the response will change and will display all the reviews and results, if you use other two options ("business_name_and_address", "phone_number"
) the response structure will be same and will return you the yelp biz_id with other details about the business.
Curl
curl -X 'POST' \
'https://x0n1-tbv3-v8eo.n7.xano.io/api:ag2Iad7F/Yelp_review_by_name_address_and_biz_id' \
-H 'Content-Type: application/json' \
--data '{"type":"biz_id","biz_id":"xKIv_b2L2pUHmsoJ-xOepg","ph_number":"","name":"","address":"","firm_city":"","firm_state":"","firm_country":""}'
Inputs
{
  "type": "business_name_and_address", // there are three types - "biz_id", "business_name_and_address", "phone_number"
  "biz_id": "xKIv_b2L2pUHmsoJ-xOepg", // if you have the biz_id
  "ph_number": "", // if you have the ph_number
  "name": "Gran Morsi", // name of the business
  "address": "22 Warren St New York, NY 10007", //Complete address
  "firm_city": "New York", //city 
  "firm_state": "NY", // This needs to be two letters other wise Yelp will throw an error for example - NY, AZ, CA etc etc etc
  "firm_country": "US" // This needs to be two letters other wise Yelp will throw an error
}
Output
type = business_name_and_address
